# CCTV 安裝自動化專案管理

本專案範例展示如何用自動化流程管理 CCTV 安裝，流程分為：

1. plan（規劃）
2. to do（待辦）
3. do（執行）
4. check（檢查）
5. sent（交付）

每個步驟都可自動推進，最終成果達到 10/10 精緻度才交付。

---

## 專案流程

- [ ] plan：需求訪談、現場評估、規劃方案
- [ ] to do：設備採購、排程、安裝前準備
- [ ] do：現場安裝、線路整理、設備設定
- [ ] check：功能測試、畫面確認、用戶驗收
- [ ] sent：產生報告、交付文件、通知客戶

---

## 自動化說明
- 可用 GitHub Actions 或本地腳本自動推進每個步驟
- 每個步驟完成後自動產生下一步待辦
- 最終自動產生交付報告
