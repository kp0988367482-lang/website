## 設計重點 (Conversion Focus)
- **行銷漏斗**：曝光 → 興趣 → 購買 → 忠誠。
- **服務方案**：三個定價等級。
- **信任建立**：成果數據 (3.8x ROAS) + 客戶見證。
- **CTA**：明確的「免費諮詢」按鈕。

## SeoulMate 文案準則
- 不從品牌類別出發，不寫「平台」「整合」「AI 指揮中心」這種空詞。
- 直接寫使用者此刻要解決的事，例如「下飛機前，先把要緊的事排好」。
- Hook 要能在 1 秒內成立，能短就短，優先使用場景、時間點、下一步。
- IG / TikTok / 搜尋 / Landing / Pricing / Checkout / Email 各自寫，不互相照抄。

# SeoulMate Conversion Prompt

- 所有文案、計畫、技能設計，請優先採用 SeoulMate Conversion Toolkit 的「4原則x10案例」結構：
  1. 以買家現場、時機、下一步為主，不用抽象名詞。
  2. Hook 必須1秒內成立，優先用場景、時間點、下一步。
  3. 各渠道（Landing、Pricing、Checkout、IG、TikTok、Search、Email）文案分開寫，不共用。
  4. 禁用「平台、整合、生態、AI指揮中心」等空泛詞，改用具體成果語言。
- 實際範例與可用 prompt 請參考：
  - skills/seoulmate-conversion-copy/SKILL.md
  - skills/seoulmate-conversion-copy/references/toolkit.md

# 高轉換 prompt 實例

- 「如果你現在腦中都是『簽證怎麼辦？房子怎麼找？』，先用這版把最基本的事弄清楚。」
- 「給已經準備行動的人。少一點摸索，多一點可以直接照著做的步驟。」
- 「如果你不想自己試錯，這版最省時間。重要決定前，先有人陪你看一遍。」

# Anti-AI 檢查

- 每句話都要能讓真實客戶說出口。
- 去除空泛品牌詞、混合比喻。
- 移除品牌名後仍要像人話。

# 實務流程

- 計畫書、手冊、技能設計都要明確標示 buyer state、copy job、hook、proof、CTA。
- 寫完後用 Anti-AI 檢查表自查。
