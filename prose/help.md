# OpenProse Help

Use this file when the user asks for `prose help` or general onboarding.

## Common Commands

- `prose help`
- `prose run <file>`
- `prose run <url>`
- `prose run <handle/slug>`
- `prose compile <file>`
- `prose update`
- `prose examples`

## Getting Started

Recommended starting points:

- a simple hello-world style program
- a small workflow using default filesystem state
- compile first if the user wants validation before execution

## Guidance

- Use filesystem state by default.
- Use in-context state only for very small programs.
- Use authoring guidance only when creating new `.prose` files.
