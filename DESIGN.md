# Design System: Authority & Growth (100/100 Standard)

## Scope Boundary
- This file defines **web visual design standards only**.
- It covers typography, color, component style, motion, and page-level UX expression.
- It does **not** define company org hierarchy, department ownership, or internal governance structure.
- Organization modeling should be maintained in dedicated docs (for example `docs/manuals/company_structure.md`).

## 1. Brand Identity: "Strategic Excellence"
- **Vibe:** 專業權威,數據驅動,信賴感,極限轉化。
- **Target:** 專業人士, 企業主, 自媒體創作者。

## 2. Visual Foundation
- **Primary Color:** #0F172A (Deep Slate - 權威底色)
- **Accent Color:** #2563EB (Indigo Blue - 技術與成長感)
- **Highlight:** #10B981 (Emerald - 成功與轉化)
- **Standard Gray:** #64748B (專業中性)
- **Background:** #F8FAFC (純淨白)

## 3. Typography (Premium Selection)
- **Headlines:** 'Outfit', sans-serif (現代感, 權威性)
- **Body:** 'Inter', sans-serif (極佳的可讀性, 蘋果風格)

## 4. UI Patterns
- **Cards:** Subtle borders (1px #E2E8F0), soft shadows (shadow-sm).
- **Buttons:** Bold corners (rounded-lg), smooth hover transitions.
- **Icons:** Phosphor Icons / Material Symbols (極簡細線風).

## 5. Audit Checklist for 100/100 Score
- [✅] **Design Audit Checklist**: Standards applied across project (no blocking issues).
