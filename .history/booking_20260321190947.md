# BookingAI 預約表單元件

本元件為「1:1 韓國規劃諮詢」預約表單，支援多步驟填寫、AI 問題建議、時段選擇與付款指引，適合獨立頁面或嵌入式使用。

---

## 功能特色
- 多步驟流程（需求填寫 → 選擇時間 → 付款 → 完成）
- Google Fonts 品牌字體、全局色彩與排版統一
- AI 輔助自動生成諮詢問題描述（可串接 Claude API）
- 支援多種諮詢主題、階段、方式與時段
- 完整表單驗證與錯誤提示
- 完成後顯示預約資訊與返回首頁按鈕

---

## 使用說明

1. **安裝依賴**
   - React 18+
   - 若需 AI 建議功能，請確保有可用的 API 端點

2. **引入元件**
   ```jsx
   import BookingAI from './BookingAI';
   // ...
   <BookingAI />
   ```

3. **自訂顏色/字體**
   - 可於 C 與 GF 變數調整品牌色與 Google Fonts

4. **API 串接**
   - AI 建議功能預設串接 anthropic Claude API，請依需求修改 fetch 目標與權限

5. **獨立頁面建議**
   - 建議將 BookingAI.jsx 作為 /booking 或 /reserve 路由的主元件
   - 可搭配 Next.js、Vite、CRA 等任意 React 專案
